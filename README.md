hsm-guide
=========

HSM documentation

### To build

- Need: maven on path
- Need: JAVA_HOME pointing to JDK
- Run: mvn docbkx:generate-pdf
- Output: target\docbkx\book.pdf

Find prebuilt book here: [book.pdf](book.pdf)
